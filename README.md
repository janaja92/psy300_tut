# Quiz Repository

## Online usage (recommended)

Repetition questions for the psy300 Statistical Learning lecture of the Data Science and Machine Learning Master's course at the University of Oldenburg. The content should primarily be accessed from the online book: [![Jupyter Book Badge](https://jupyterbook.org/badge.svg)](https://janaja92.github.io/psy300_tut/)
